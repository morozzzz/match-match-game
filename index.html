How to play

Memory is a counter game where the object is to find pares. When the game begins, all pictures are hidden.

To play

  • Select two cards to try to math the pictures.
  • If you match the pictures you can go again.
  • If they don't match it is the computer turn them.
  • The player that finds all pairs wins!
  • Have fun!
